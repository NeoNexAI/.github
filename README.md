## 🇪🇸 ¿Qué es este repositorio?

Este es el repositorio **especial `.github`** de la organización. GitHub lo trata de forma diferente: lo que pongas aquí se aplica **por defecto a todos los repos** de `@neonexai` que no tengan su propia versión. Lo escribes una vez → vale para toda la organización.

Contiene dos cosas:

1. **El perfil público de la organización** ([`profile/README.md`](profile/README.md)) — lo que se muestra en [github.com/neonexai](https://github.com/neonexai).
2. **Los archivos de salud de la comunidad** (_community health files_) — plantillas y políticas heredadas por todos los repos.

## 🇬🇧 What is this repository?

This is the organization's special **`.github` repository**. GitHub uses it to provide **default community health files and the public org profile** across every `@neonexai` repository that doesn't define its own. Write once → applies org-wide.
