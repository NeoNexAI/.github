# Assets del perfil

Coloca aquí el logo de la organización con el nombre exacto:

```text
neonex-logo.png
```

Los README (`profile/README.md` y el `README.md` raíz) ya apuntan a:

```text
https://raw.githubusercontent.com/neonexai/.github/main/profile/assets/neonex-logo.png
```

En cuanto subas `neonex-logo.png` a esta carpeta y se fusione a `main`, el logo
aparecerá automáticamente en el perfil de la organización.

> Recomendado: PNG con fondo transparente, ~400–600 px de ancho. Si prefieres
> SVG, súbelo como `neonex-logo.svg` y avísame para actualizar las referencias.
