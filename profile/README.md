## 🧭 Filosofía

1. **Sin humo IA.** Construimos sistemas que funcionan, que decides tú, y que son tuyos.
2. **Sin lock-in.** Todo el código entregado a clientes es de su propiedad. Filosofía portable.
3. **OpenSource first.** Lo que pueda ser público, lo es. Devolvemos al ecosistema.
4. **Karpathy principles** — Think before coding · Simplicity · Surgical changes · Goal-driven.
5. **Revenue-first.** Generamos valor real antes de construir infraestructura especulativa.
